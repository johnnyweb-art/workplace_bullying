🤖 LLM 語意複核（選用 · 進階，需自備模型）

這是選用的進階功能。一般使用不需要它——「② 會議輸入 → ③ 風險判讀」的規則判讀在手機上即可免費、即時運作。本功能是再用大型語言模型做一次語意複核，補抓隱性語氣、降低誤判並給管理建議。

使用前須知（這裡沒跑出結果，不是故障）：
・本功能需自備模型端點與 API 金鑰，未設定就會無法執行，屬正常現象。
・📱 手機無法使用「本機 Ollama」（手機沒有在本機跑模型），且雲端 API 會被瀏覽器安全機制（CORS）阻擋——手機要用，請改走「代理（proxy）」，設定方式請洽本工具建置者。
・💻 電腦端可在本機安裝 Ollama 免費使用；雲端 API 則需付費金鑰（用多少付多少）。
⚠ 啟用後會把逐字稿傳送到你設定的端點，請先去識別化（移除姓名），或使用本機模型以保護隱私。

5 大構成要件自我檢核

勞動部《職場霸凌防治準則》：原則上須同時符合下列 5 要件才構成霸凌（情節重大者不以「持續性」為必要）。逐項檢視可協助你判斷自身行為是否落入風險區。

法律定義（職安法第 22 條之 1）

本法所稱職場霸凌，指勞工於勞動場所執行職務，因其事業單位人員利用職務或權勢等關係，逾越業務上必要且合理範圍，持續以冒犯、威脅、冷落、孤立、侮辱或其他不當之言詞或行為，致其身心健康遭受危害。但情節重大者，不以持續發生為必要。
